# Digital Talent Management System

## Project Overview

Digital Talent Management System is a web application designed to manage users, authentication, and future task workflows.

This repository currently includes Sprint 1 implementation.

## Tech Stack

Frontend:
- React.js

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Authentication:
- JWT (JSON Web Token)

## Sprint 1 – Foundation & Authentication

Completed Features:

- Project setup (Frontend + Backend architecture)
- MongoDB database connection
- User Registration API
- User Login API
- JWT-based authentication
- Protected routes
- GitHub repository setup

  ## How to Run the Project Locally
  step 1 : Clone the Repository
  
  step 2 : Backend Setup
  
  cd backend
  npm install
  npm start
  server runs on :  http://localhost:5000
  
  Step 3 : Frontend Setup
  
  cd frontend
  npm install
  npm start
  
  Frontend runs on: http://localhost:3000
